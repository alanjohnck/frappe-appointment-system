#  Appointment Booking System (Frappe Framework)

A multi-business appointment booking system built using the **Frappe Framework**.

##  Features
 Multi-business setup (each business has separate services, staff, and timezone)  
 Appointment booking, rescheduling, and cancellation  
 Staff & service management  
 Backend validation (prevents double-booking)  
 Admin dashboard for each business  
 (Upcoming) WhatsApp + AI intent detection using Twilio  
 (Upcoming) Docker-ready deployment
